Ejercicio 5

Servidor Web simple en node.js. Contador de visitas de página web. Se puede acceder a las distintas secciones cambiando la url, sirve cualquier url dentro de la web, que creará una sección (ej: 'url:8080/seccion1'). Muestra un contador de visitas de esa sección. Si se accede a la página principal (sin sección) muestra un contador de todas las secciones.

  • Utiliza el módulo "fs" para leer 2 plantillas de documento html para componer la respuesta en html: la cabecera y el pie, entre los mismos se mostrará el resultado
  • Crea un servidor web simple escuchando en el puerto 8080
  • Al recibir una petición comprueba la sección (a continuación de la url) y añade un contador a esa sección
  • En cada sección muestra en json las peticiones recibidas (sección, nº de petición de sección, nº petición general, ip del cliente y fecha y hora de petición). En la sección principal mostrará las peticiones de todas las secciones
